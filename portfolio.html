Portfólio

Todos os meus projetos se encontram em meu Github.

Aqui estão alguns deles:
  • Quiz - Inspirado nos testes do BuzzFeed. Todo o conteúdo é obtido de um arquivo JSON, logo é possível criar diferentes testes.
  • The Blog - Blog com notícias personalizáveis. Cada notícia é um componente, tornando fácil sua edição.
  • Pokédex - Na listagem é possível ver os Pokémons a partir de sua respectiva geração, é listado o nome, número, tipos e também uma foto de cada um deles.
